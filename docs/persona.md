# 客户画像

## 画像：李姐

```
年龄: 38-55 岁女性
地域: 二三线城市（有医院但优质医疗资源有限）
教育: 大专/本科
职业: 教师/公务员/家庭主妇，有一定空闲时间
手机: 中端安卓，微信重度用户，会用小程序但不擅长折腾 App
```

## 一天的生活

| 时间 | 行为 |
|------|------|
| 6:30 | 被足底疼醒，拖了 10 分钟才下床 |
| 7:00 | 吃早饭没啥胃口，喝半碗粥 |
| 8:00-17:00 | 上班，坐着还好，站起来走路脚就疼 |
| 12:00 | 午饭吃不多，同事问"你怎么脸色这么差" |
| 22:00 | 躺床上，脑子里想着复查结果，睡不着 |
| 凌晨 1-2 点 | 终于迷迷糊糊睡着 |

## 核心痛点（按优先级）

### 1. "查了也查不出原因"
> 化验单上一堆箭头，医生说"没什么大问题，回去观察"。但她每天真的难受。

- 去了 3 家医院、看过 5 个科室
- 医生给 5 分钟就开药，没时间听她说完
- 药吃了没效果，不知道是药不对还是需要时间
- **核心焦虑：我到底得了什么病？还是我想多了？**

### 2. "跟家人说不清楚"
> 老公说"你就是太累了多休息"，她觉得不被理解。

- 家人没有经历过，无法共情
- 反复解释症状很累，干脆不说了
- 想找"同病相怜"的人聊聊，但没有渠道

### 3. "去复查不知道怎么说"
> 医生问"最近怎么样"，她只能说"还行吧"，白白浪费就诊机会。

- 记不住过去两周症状变化的细节
- 忘了问准备好的问题
- 就诊像赶场，出来才想起该问的没问

### 4. "不知道什么有用什么没用"
> 试了补铁剂、中药、按摩、热敷……记不清哪个有效了。

- 没有系统记录，靠记忆判断
- 今天觉得好一点就停了，过几天又复发
- 想找到规律，但脑子记不住

## 使用产品的典型场景

| 场景 | 触发 | 行为 |
|------|------|------|
| **睡前记录** | 被提醒或自己想起来 | 花 30 秒快速记今天的症状严重度 + 睡眠 |
| **就诊前** | 明天要去复查 | 导出报告，整理想问的问题 |
| **难受时** | 刚发作完 | 打开 App 看一眼趋势，"果然每次熬夜第二天就加重" |
| **深夜刷论坛** | 睡不着 | 看有没有相似症状的新帖子，回复一下 |

## 她对产品的要求

| 要求 | 优先级 |
|------|--------|
| **操作简单** — 最好 30 秒内完成一次记录，不要填表单 | 🔴 |
| **看得懂** — 不要医学术语，用普通人能理解的语言 | 🔴 |
| **有安全感** — 数据是我的，我不是小白鼠，分析要说明依据 | 🔴 |
| **能找到人** — 有跟我症状相似的人在讨论 | 🟡 |
| **能帮到我就诊** — 生成的报告医生也看得懂 | 🟡 |
| **不打扰** — 提醒适度，别弹一堆通知 | 🟢 |

## 竞品对比：为什么她不用现有产品

| 产品 | 她试过吗 | 问题 |
|------|---------|------|
| 健康手环/手表 | 买过 | 只有心率/步数，不知道她的贫血和乏力 |
| 月经/备孕 App | 用过 | 不相关，她已经过了那个阶段 |
| 好大夫/丁香医生 | 用过 | 问一次几十块，问完就没了，不跟踪 |
| 百度/小红书 | 经常 | 越搜越害怕，全是焦虑内容 |
| 微信病友群 | 加过 | 消息太多太杂，有用的信息被淹没 |

---

## 一句话总结

> **李姐不需要另一个健康 App。她需要一个帮她把"说不清的不舒服"变成"医生听得懂的描述"，同时让她知道自己不是一个人的工具。**
